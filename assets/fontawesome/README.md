# Font Awesome Directory

Ten katalog zawiera lokalne pliki Font Awesome 6.5.1 dla aplikacji AEP.

## Wymagana struktura

```
fontawesome/
├── css/
│   └── all.min.css
└── webfonts/
    ├── fa-brands-400.woff2
    ├── fa-brands-400.ttf
    ├── fa-regular-400.woff2
    ├── fa-regular-400.ttf
    ├── fa-solid-900.woff2
    └── fa-solid-900.ttf
```

## Instrukcje pobierania

Zobacz plik `/FONTS_OFFLINE_SETUP.md` w katalogu głównym projektu.

Pobierz Font Awesome Free z: https://fontawesome.com/download
